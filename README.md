# Bank Customer Churn Prediction 📉

## Objective
To build a model that identifies customers who are likely to exit the bank based on factors like Credit Score, Geography, Gender, Age, and Balance.

## Key Instructions Followed
- **Data Preparation:** Cleaned the 'Churn_Modelling' dataset.
- **Categorical Encoding:** Used Label/One-Hot encoding for Geography and Gender columns.
- **Machine Learning Modeling:** Trained a classifier (Logistic Regression or Random Forest).
- **Feature Importance:** Analyzed which factors (like Age or Balance) influence a customer's decision to leave.

## Skills Demonstrated
- Data Preprocessing & Encoding
- Supervised Machine Learning
- Interpretation of Feature Importance
